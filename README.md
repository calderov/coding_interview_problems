# Coding Interview Problems and Solutions

This collection of problems is a good source of study material for coding interviews.

Some of the exercises for the course come from:
1. [Grokking the Coding Interview: Patterns for Coding Questions](https://designgurus.org/course/grokking-the-coding-interview). 
2. [Leetcode](https://leetcode.com)
3. [Neetcode 150](https://neetcode.com)
4. Youtube
5. Known "most asked questions" for Meta, Amazon and Microsoft

as well as some problems of my own creation.

I've included code for the proposed solutions, notes and references for the following topics:

1. Pattern: Two Pointers
1. Pattern: Fast & Slow Pointers
1. Pattern: Islands (or Matrix Traversal) (*NEW*)
1. Pattern: Sliding Window
1. Pattern: Merge Intervals
1. Pattern: Cyclic Sort
1. Pattern: In-place Reversal of a LinkedList
1. Pattern: Tree Breadth First Search
1. Pattern: Tree Depth First Search
1. Pattern: Two Heaps
1. Pattern: Subsets
1. Pattern: Modified Binary Search
1. Pattern: Bitwise XOR
1. Pattern: Top 'K' Elements
1. Pattern: K-way Merge
1. Pattern: Topological Sort (Graph)
1. Pattern: Multi-threaded (*NEW*)

The included code is written in Python, although a C++ version of the solutions may come in the future.
